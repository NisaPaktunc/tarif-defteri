# 🍳 Tarif Defterim

Modern ve kullanışlı bir tarif yönetim uygulaması. Tariflerini ekle, düzenle, puanla ve kategorilere göre filtrele!

## 🚀 Canlı Demo
🔗 [Uygulamayı Görüntüle](https://nisanin-tarif-defteri.netlify.app)

## 📸 Ekran Görüntüsü
![Tarif Defterim](./screenshot.png)

## ✨ Özellikler
- ➕ Tarif ekleme
- 📋 Tarifleri listeleme
- ✏️ Tarif güncelleme
- 🗑️ Tarif silme
- ⭐ Yıldızla puanlama
- 🔍 Tarif arama
- 🏷️ Kategoriye göre filtreleme
- 💾 LocalStorage ile veri saklama

## 🛠️ Kullanılan Teknolojiler
- [React](https://reactjs.org/) - UI kütüphanesi
- [Vite](https://vitejs.dev/) - Build aracı
- [Tailwind CSS](https://tailwindcss.com/) - CSS framework

## ⚙️ Kurulum
```bash
# Projeyi klonla
git clone https://github.com/kullanici-adin/tarif-defteri

# Klasöre gir
cd tarif-defteri

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```
